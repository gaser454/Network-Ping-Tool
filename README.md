# 📶 Network Ping Tool

A simple network diagnostic utility implemented in multiple programming languages.

---

## 📖 Overview

This repository demonstrates how to execute network ping requests using:

- JavaScript
- Python
- C#
- PHP

Useful for:

- Network diagnostics
- Server monitoring
- Connectivity testing
- Infrastructure learning

---

## ✨ Features

- Ping remote hosts
- Console-based output
- Multi-language implementation
- Beginner-friendly examples

---

```text
javascript/

```

## Example Output

```text
PING google.com (142.250.74.14): 56 data bytes

icmp_seq=0 ttl=117 time=23ms
```

## Running

### JavaScript

```bash
node ping.js
```

### Python

```bash
python ping.py
```

### C#

```bash
dotnet run
```

### PHP

```bash
php ping.php
```
